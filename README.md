# TimeTravel Agency - Eve MOISSON et Thomas HERICHARD

TimeTravel Agency est une webapp immersive créée dans le cadre d’un projet pédagogique autour d’une agence de voyage temporel fictive.  
L’objectif du projet est de proposer une expérience utilisateur moderne et interactive permettant de découvrir plusieurs destinations temporelles, d’échanger avec un assistant virtuel et de simuler une réservation.

## Aperçu du projet

La webapp met en scène trois destinations principales :

- **Paris 1889**
- **Crétacé -65M**
- **Florence 1504**

L’interface a été pensée pour offrir une ambiance premium et immersive, avec un univers visuel sombre, des cartes interactives, des pages dédiées à chaque époque et un chatbot simulant un assistant de voyage temporel.

## Stack technique

Le projet a été développé avec des technologies front-end simples et accessibles :

- **HTML5**
- **CSS3**
- **JavaScript**

## Features implémentées

Les fonctionnalités actuellement développées dans la webapp sont :

- Landing page immersive
- Hero section avec univers visuel futuriste
- Présentation de l’agence
- Galerie interactive des 3 destinations
- Pages dédiées pour :
  - Paris 1889
  - Crétacé
  - Florence 1504
- Widget chatbot flottant
- Réponses simulées aux questions fréquentes
- Conseils sur les destinations
- Informations tarifaires fictives
- Section de réservation avec aperçu dynamique
- FAQ intégrée
- Design responsive

## Fonctionnement du chatbot

Le chatbot joue le rôle d’un assistant virtuel de **TimeTravel Agency**, une agence de voyage temporel de luxe.

Il peut répondre à des questions sur :

- les destinations
- les prix
- le choix de l’époque
- le fonctionnement de l’agence
- la réservation
- la sécurité des voyages

Le chatbot fonctionne actuellement avec une logique simulée en JavaScript, à partir de réponses prédéfinies selon les mots-clés saisis par l’utilisateur.

## Outils IA utilisés

Dans une logique de transparence, plusieurs outils d’IA générative ont été utilisés au cours du projet :

- **ChatGPT** : aide à la conception de l’interface, rédaction de contenu, génération et adaptation du code
- **Outil de vibe coding / génération assistée** : utilisé pour réfléchir à la structure initiale de la webapp
- **IA générative visuelle** : possible utilisation dans le projet précédent pour la création ou l’inspiration des visuels temporels

L’IA a été utilisée comme **outil d’assistance à la création**, mais les choix de structure, d’intégration, de design et d’adaptation ont été réalisés dans le cadre du projet pédagogique.
